Strassen Matrix Multiplication

Steps of Strassen's Algorithm

For two 2x2 matrices, A and B, Strassen’s algorithm divides them into smaller 2x2 submatrices, reducing the multiplications to only 7 instead of 8.

  1. 1. Split each matrix A and B into four submatrices:
    2. 
    A = | A11 A12 |     B = | B11 B12 |
        | A21 A22 |         | B21 B22 |
  2. 2. Calculate the following 7 products:
    3.     M1 = (A11 + A22) * (B11 + B22)
    M2 = (A21 + A22) * B11
    M3 = A11 * (B12 - B22)
    M4 = A22 * (B21 - B11)
    M5 = (A11 + A12) * B22
    M6 = (A21 - A11) * (B11 + B12)
    M7 = (A12 - A22) * (B21 + B22)
  3. 3. Use the products to build the result matrix C:
    4.     C11 = M1 + M4 - M5 + M7
    C12 = M3 + M5
    C21 = M2 + M4
    C22 = M1 - M2 + M3 + M6
  4. 4. Combine C11, C12, C21, and C22 to form the final 2x2 result matrix:
    5. 
    C = | C11 C12 |
        | C21 C22 |
